1. 重启应用: `npm run dev`
2. 访问设置页面: http://localhost:3000/settings
3. 在模型下拉框中查看全部 60+ 模型
4. 选择 **Claude Opus 4.5** 或 **Grok Code Fast 1**
5. 开始创建你的网站！
